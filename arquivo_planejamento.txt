EXPERIÊNCIAS:

	- PROJETOS DE CIÊNCIA DE DADOS
		
		* Recomendação de Produtos: Desenvolvemos um sistema de recomendação voltado para clientes que já possuem ao menos um produto no banco, com foco na verticalização do relacionamento. 
		                            Devido à complexidade do sistema bancário, selecionamos clientes sem comprometimento de crédito, buscando garantir consistência nas sugestões.
									Para tornar as recomendações mais precisas, coletamos diversos atributos para uma comparação justa. Usamos informações cadastrais — como estado, 
									faturamento e dívida de mercado — e dados bancários, como número de parcelamentos de empréstimos, quantidade de seguros e outras variáveis relevantes.
									A abordagem técnica incluiu um pré-processamento personalizado, visando uma vetorização eficiente. Com isso, aplicamos dois métodos de recomendação:
										- Filtragem colaborativa
										- Filtragem por similaridade do cosseno
									Após a construção do modelo, implementamos uma camada de refinamento de produtos. Por exemplo, não faz sentido recomendar empréstimos a clientes com bloqueio de proposta. 
									Esse filtro garante uma recomendação precisa e alinhada ao perfil do cliente. O produto final inclui:
										- Um dashboard operacional
										- Integrações com canais digitais (app e internet banking – em implantação)
										- Ferramentas para gerentes comerciais (também em implantação)

									Ferramentas Utilizadas: SAS Viya, SAS Visual Analytics, Python e AWS S3.

		* Modelo de Churn de Clientes: Historicamente, o banco compreendia o abandono de produtos específicos pelos clientes, mas não havia uma visão clara sobre a saída total do cliente do ecossistema bancário. 
									   Para suprir essa lacuna, desenvolvemos — em parceria com outros cientistas de dados — um sistema baseado em machine learning para prever o churn no mês seguinte.
									   Identificamos os sete indicadores mais estratégicos para o nosso contexto bancário, analisando suas tendências de alta ou queda. A partir disso, construímos sete modelos preditivos, 
									   cuja combinação gera uma pontuação final indicando a probabilidade de churn. Na parte técnica, utilizamos o algoritmo Gradient Boosting Classifier, que alcançou resultados expressivos após refinamentos:
											- Precisão média: 85%
											- Recall: 88%
											- AUC: 90%
									   Essas previsões possibilitaram ações direcionadas para retenção de clientes. O projeto resultou em:

											- Um dashboard operacional para acompanhamento dos clientes com alto risco de churn
											- Suporte para decisão estratégica por superintendentes, gerentes regionais e comerciais
										
									   Ferramentas Utilizadas: SAS Viya, SAS Visual Analytics, Python e AWS S3.
